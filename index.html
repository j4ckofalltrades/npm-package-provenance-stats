This site shows the top 500 most-downloaded packages on npm showing which have been uploaded with attestations.

Packages marked in green have attestations for their latest version.

Generating provenance statements

npm supports generating provenance statements for the packages you publish.

This allows you to publicly establish where a package was built and who published a package, which can increase supply-chain security for your packages.

Refer to the npm Docs for more details.
